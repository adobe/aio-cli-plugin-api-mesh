# Usage
```sh-session
$ aio plugins:install -g @adobe/aio-cli-plugin-PLUGINNAME
$ # OR
$ aio discover -i
$ aio PLUGINNAME --help
```
Local Development
```
aio plugins:link commerce-gateway/tenant
```
